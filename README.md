## Overview
Animated monthly sales dashboard built with **Microsoft Fabric Notebooks**.  
Features dynamic line and bar chart animations with KPIs including total, average, max, and min sales.

---

## Features
- **Line Chart:** Shows monthly sales trends over time  
- **Bar Chart:** Displays actual monthly sales values  
- **Dynamic KPIs:** Total, Average, Max, and Min updated dynamically  

---

## Files
- `dataset/monthly_sales.csv` – Input dataset  
- `gifs/monthly_sales_dual_animation.gif` – Pre-generated animation  
- `notebooks/Monthly_Sales_Animation.ipynb` – Notebook to run and generate GIF  

---

## How to Run
1. Open the notebook `Monthly_Sales_Animation.ipynb` in **Microsoft Fabric**  
2. Ensure the dataset `monthly_sales.csv` is in the `dataset/` folder  
3. Run all cells  
4. The GIF will be generated in `gifs/` and displayed inline
