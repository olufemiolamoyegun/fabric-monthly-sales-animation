## Overview
Animated monthly sales dashboard built with Microsoft Fabric Notebooks.  
Displays line and bar chart animations with dynamic KPIs (total, average, max, min).

---

## What it does
- **Line chart**: Shows monthly sales trends  
- **Bar chart**: Displays monthly sales values  
- **Dynamic KPIs**: Total, average, max, and min  

---

## Files
- `dataset/monthly_sales.csv` – sample sales data  
- `gifs/monthly_sales_dual_animation.gif` – generated animation  
- `notebooks/Monthly_Sales_Animation.ipynb` – main notebook  

---

## How to run
1. Open `Monthly_Sales_Animation.ipynb` in Microsoft Fabric.  
2. Ensure the dataset `monthly_sales.csv` is available in `dataset/`.  
3. Run all cells.  
4. The GIF will be generated in `gifs/` and displayed inline.
